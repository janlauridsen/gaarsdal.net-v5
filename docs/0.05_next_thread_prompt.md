# Next Phase – Thread Kickoff Prompt

Vi har defineret et AI-baseret fagligt afklaringssystem,
hvor UI + server fungerer som en state machine,
AI er stateless og kontrolleret,
og tone, etik og længde styres af state.

Artefakter er dokumenteret (system overview, state machine,
tone-kontrakt, guard-state, UX review, testspørgsmål).

Næste fase:
- omsætte state machine til konkret backend-logik
- definere API-kontrakter mellem UI og server
- beslutte præcis hvor AI kaldes og hvor den ikke gør
- evt. prototype én fuld vertikal (fx flyskræk)

Fortsæt uden at gentage grundprincipperne.
