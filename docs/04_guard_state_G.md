# Guard State (G)

Formål:
Håndtere mobning, sarkasme, provokation, destruktiv eller pervers adfærd.

Psykologisk grundlag:
- Grænsesætning
- Grey Rock
- Ekstinktion

AI-rolle:
- Kun detektion / klassifikation

Respons:
- Fast
- Kort
- Ikke-eskalerende
- Ingen forklaring
- Ingen variation

Gentagelser giver samme eller kortere svar.

G-state kan afbryde alle andre states.
