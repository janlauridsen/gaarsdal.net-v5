# Worst-case UX Review

Dette dokument identificerer situationer hvor korrekt systemadfærd
kan opleves negativt af brugeren.

Kerneindsigt:
- UX-problemer opstår ofte ved mismatch mellem forventet dialog
  og leveret faglig afklaring.

Designkorrektioner:
- Afslut tidligere
- Brug færre ord
- Gør rammer tydeligere
- Undgå gentagne afgrænsninger

Mål:
At reducere irritation uden at øge empati, dialog eller hukommelse.
