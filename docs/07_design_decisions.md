# Design Decisions

Bevidste valg:
- Ikke chatbot
- Ingen samtalehukommelse
- Ingen adaptiv empati
- Ingen personlig vurdering
- AI analyseres evt. post-output for etik og tone

Fravalg:
- Personalisering
- Session-baseret hukommelse
- Relationel dialog

Rationale:
Systemet skal være auditérbart, forudsigeligt og etisk robust.
