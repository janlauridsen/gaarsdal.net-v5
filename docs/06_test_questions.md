# Test Questions – Faglig afklaring om hypnoterapi

Dette dokument indeholder 100 testspørgsmål designet til at:
- validere state machine-logikken
- teste emne-arbitrage og afklaring
- afdække UX-friktion
- stressteste grænser (etik, rådgivning, G-state)
- sikre konsistent tone og afslutning

Spørgsmålene er arrangeret så de “zoomer ind”:
Alternativ behandling → hypnose → evidens → risiko → praksis → flyskræk.

De er **ikke alle beregnet til at blive besvaret**.
Nogle skal:
- omdirigeres
- afvises
- afsluttes
- håndteres i G-state
